# QuizApp
*Created as part of Udacity's Android Basics by Google Nanodegree Program*
_______________________

#### Project Overview


The goal was to create an educational app that quizzes a user about a certain topic.
I chose Java and Android as the topic for this app. The app presents 7 quiz questions (based on the quiz level specified by user - 
Easy/Medium/Difficult) one by one. The questions may ask to choose one answer from a group of radio buttons, multiple answers 
from a group of check boxes, or type in an answer. The questions can be navigated using PREV and NEXT buttons, and answers can be 
changed before the answers are submitted (using SUBMIT button that appears on the last question). Upon submitting the answers, the score 
is calculated and displayed along with the correct/incorrect answers.
